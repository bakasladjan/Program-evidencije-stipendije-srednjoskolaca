# Program evidencije stipendije srednjoskolaca
Višeslojna veb aplikacija razvijena u C#/.NET okruženju sa SSMS bazom podataka, implementiranom poslovnom logikom, servisnim slojem i korisničkim interfejsom na srpskom jeziku. Projekat obuhvata autentifikaciju korisnika, rad sa podacima, validacije, štampu izveštaja, primenu OOP i SOLID principa i kompletnu tehničku i korisničku dokumentaciju.
